# 📖 [Awesome MarketPlace] <a name="about-project"></a>

> Describe your project in 1 or 2 sentences.

**[Awesome MarketPlace]** is a RESTful API for an online marketplace that allows users to buy and sell products, 
manage their inventory and process orders.

### Key Features <a name="key-features"></a>

> MarketPlace is a microservice which  has 4 Services(Authentication: for user authentication, products: has a way a user create and update the goods on the system, orders and notification for notifying a user)

- **Authentication**
- **Products**
- **Ordering**
- **Notification**

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: a laptop and Ensure you have Node.js and docker

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/freddykaberuka/marketPlace.git
```

### Install

Install this project with:


```sh
  cd marketPlace
```

### Usage

To run the project, execute the following command:


